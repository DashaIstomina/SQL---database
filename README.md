# Проект работы с реляционной базой данных

В данном проекте представлены примеры решения SQL-задач с использованием:  
- группировки данных и агрегатных функций;  
- объединения таблиц (JOIN);  
- подзапросов;  
- общих табличных выражений (CTE);  
- оконных функций;  
- условных конструкций (CASE);  
- функций работы с датами и временем;  
- представлений (VIEW);  
- индексов (INDEX).
